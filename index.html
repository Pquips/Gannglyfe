Blinking Light

To make the light blink ten times I made the "i=0" as a variable. I then put a "while" statment that said while "i" is less than 5, when it will wait 0.5 seconds (time.sleep) then it powers the led on again. Internally everytime the led is turned on it will add 0.5 to the variable "i" (i=i+0.5) thus making the led blink 10 times.

Led Keyboard

This code makes it so that when I click an assigned button on the keyboard of a computer than instantly it will trigger an led on. Using the Tkinter plugin on the Raspberry Pi that adds thing like "keyup" and "keydown" functions. I configure 6 keys to light up 3 lights at a time. They can all be on at the same time as long that the keys are held down. This counts as the user input challenge and the lightshow challenge because we can play along to any tune.
